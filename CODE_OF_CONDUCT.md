# Code of conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful and constructive. Harassment, personal attacks, and disruptive behavior aren't tolerated in issues, pull requests, or any other project space.

Reports of unacceptable behavior can be sent to [security@teamretro.com](mailto:security@teamretro.com). Maintainers will review and respond, and may edit, remove, or reject contributions and contributors that violate this code.
