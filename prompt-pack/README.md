# Prompt pack — the same practice, any AI agent

No skill support needed. Works in Cursor, GitHub Copilot, Codex, Gemini CLI, a chat window — anywhere you can paste a prompt.

## Use

1. At the end of a working session, paste [`session-retro-prompt.md`](session-retro-prompt.md) into the conversation.
2. Review the entry it produces (you were there too — correct anything wrong).
3. Save it to `docs/ai-retros/entries/YYYY-MM-DD-<slug>.md` in your repo and commit it.
4. Before your team retro, paste [`retro-brief-prompt.md`](retro-brief-prompt.md) with your accumulated entries.

Optionally add the session-retro prompt to your agent's standing rules (`.cursor/rules`, `AGENTS.md`, custom instructions) so "run a retro on this session" works without pasting.
